Primero, construyamos una tabla que contiene el precio promedio de cada corte

precio_prom <- diamantes |> 
  group_by(corte) |> 
  summarise(
    precio_prom = mean(precio)

En label se establece una etiqueta de la tabla, ¿esto se contrapone a #| label:?

En format.args se pone separador de coma a los números, pero debe ponerse la función list(), los argumentos provienen de format

En escape = TRUE permite añadir escapes (insertar caracteres especiales)

precio_prom |> 
  kableExtra::kbl(
    format = "html",
    digits = 2,
    col.names =  c("Precio promedio", "Corte"),
    align = c("l", "r"),
    caption = "Título de la tabla",
    format.args = list(big.mark = ","),
    booktabs = TRUE
  )
