Código computacional

Con el propósito de propiciar la transparencia y la reproducibilidad de los resultados presentados en esta investigación, se pone a disposición el código fuente desarrollado en R, versión 4.5.1 (R Core Team, 2025). Dicho proyecto permite reproducir en su totalidad el procesamiento de los datos, las estimaciones econométricas, así como las figuras empleadas a lo largo de este documento. En esta sección, se describen los principales elementos necesarios para su utilización.

En primer lugar, el código fuente puede consultarse en el siguiente repositorio: https://drive.google.com/drive/folders/16GR9Nv6JlYYmLo0ICuJ1hxYOaj2-kiNU?usp=drive_link

Para ejecutar el código por vez primera, se sugiere seguir los siguientes pasos:

  1. Abrir tesis.Rproj1

  2. Ejecutar en la consola la siguiente instrucción

```{r}
# install.packages("renv")
library(renv)
renv::restore()
```

De este modo, se restaurará el entorno computacional empleado durante el desarrollo de esta investigación, incluyendo las versiones específicas de los paquetes utilizados. En ese sentido, puede utilizarse la función renv::diagnostics() para verificar el estado de dicho entorno computacional.

```{r}
renv::diagnostics()
```

En términos generales, el flujo de trabajo del presente proyecto computacional puede resumirse como sigue:

datos/import -> script_1 -> datos/export -> scripts_restantes -> figuras | datos/export

Se observa que cada script realiza una tarea específica y genera archivos intermedios que posteriormente son utilizados por los siguientes scripts. Esta organización busca facilitar la revisión del código, evitar la duplicación de procesos, así como favorecer la transparencia y reproducibilidad de los resultados obtenidos.

Por tanto, las personas interesadas en reproducir los resultados de la presente investigación deben ejecutar los scripts descritos líneas arriba. Mientras que, aquellos únicamente interesados en verificar los resultados reportados pueden consultar el archivo datos_procesados.xlsx, que contiene los datos utilizados para construir los cuadros y figuras presentados en esta investigación.
